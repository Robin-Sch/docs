title: nmap
---

## Finding generic OS details
`nmap -O ip`

## Finding all hosts in network
`nmap -n 192.168.0.0/24`

## Scripts
```bash
sudo updatedb
sudo locate *.nse
nmap --script-help <path to script> # get help about this script
nmap --script <path to script> <target> # run script
```

## Other
`nmap -sT ip` simple TCP scan
`nmap -sU ip` simple UDP scan
`nmap -sV -p- ip` scan for all tcp ports and detect service
`nmap -p- --script=banner ip` scan for all tcp ports and detect service
`nmap -n -PN -sT -sU -p- ip` scan for all tcp and udp ports
